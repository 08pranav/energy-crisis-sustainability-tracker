# Energy Crisis & Sustainability Tracker (SDG 7 + SDG 13)

Production-grade MERN mini-project scaffold for CEP-focused full stack development.

## Tech Stack
- Frontend: React + Redux Toolkit + React Router + Recharts + D3 + Tailwind CSS
- Backend: Node.js + Express + JWT + bcrypt + Joi + WebSocket + node-cron
- Database: MongoDB + Mongoose
- Data: IEA API + BP CSV ingestion
- Deployment: Docker + Render/Railway (backend) + Vercel (frontend)

## Repository Layout
- backend: API server, ingestion, auth, models, jobs, websocket
- frontend: React app, Redux slices, auth UI, visualization components
- docs: architecture handbook, endpoint contracts, CEP mapping, workflow

See docs/ARCHITECTURE_BLUEPRINT.md for full details.
